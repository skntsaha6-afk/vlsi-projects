# DRC Analyzer — OOP Based
Analyzes Calibre/Synopsys DRC reports using Python OOP.
Built as part of VLSI + Agentic AI learning journey.

## Features
- Extracts DRC violations from report files
- Counts violations by rule name
- Identifies most common violation

## Usage
python drc_analyzer.py

## Tech
Python 3.11 | OOP | Counter | Type hints
